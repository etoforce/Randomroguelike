ПЕЩЕРНЫЙ РОГАЛИК

Запуск:
1. Открой index.html в современном браузере.
2. Для первого знакомства выбери «Обучение».
3. Для обычного прохождения нажми «В путь!» и выбери класс.

Управление:
- Стрелки клавиатуры или кнопки под картой — движение и поворот.
- Действия с врагами, сундуками и другими объектами появляются справа.

В этой версии:
- отдельный учебный этаж из 7 шагов;
- адаптивная раскладка для широких и узких экранов;
- более заметные подсказки по управлению;
- обычный режим и сохранения не смешиваются с обучением.
